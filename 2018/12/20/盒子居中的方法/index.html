HTML

盒子居中的方法：

1.宽度和高度已知的。

思路：
父元素相对定位
子元素绝对定位

子元素{

left: 50%;top: 50%;
margin-left: 减去子元素的宽度一半。
margin-top: 减去子元素的高度一半；

}

2.宽度和高度可以随意改变

思路：
父元素相对定位
子元素绝对定位

子元素{

left: 0;top: 0; bottom:0;right:0;

margin:auto;

}

3.flex布局
4.平移 定位+transform
